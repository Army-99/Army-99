MyMonitorly — SaaS Portfolio Intelligence  [💻Live EN](https://mymonitorly.com/en) [💻Live IT](https://mymonitorly.com/it)

SaaS web app for retail investors to track, analyse and receive alerts on hybrid portfolios (crypto, stocks, ETFs, bonds) in one place — enhanced by institutional-grade on-chain metrics and personalised AI reports.

● Built dedicated background hosted services covering real-time price ingestion, technical indicators, Bitcoin on-chain analytics, AI report generation, alert evaluation and third-party API synchronisation.

● Implemented real-time price, technical and time-based alerts delivered via SignalR WebSocket and Telegram Bot, with Stripe subscription handling and AWS SES transactional email delivery.

● Integrated multiple external data sources: CoinGecko, CoinMarketCap, Zerion, Dune Analytics, EODHD, FMP, Finnhub, OpenRouter (Claude/GPT), NewsAPI and Google Search Console.

● Delivered a mobile-first frontend with Zustand, React Query and react-i18next (IT/EN), backed by an ASP.NET Core API following Repository + Unit of Work architecture with JWT authentication.

● Published a separate Next.js landing page (SSG/SEO) and a dedicated React + Vite dashboard app as two distinct frontend surfaces.

Tech stack:
FE App: React, TypeScript, Vite, Zustand, React Query, react-i18next
Landing: Next.js (SSG / SEO)
BE: ASP.NET Core, SignalR, Entity Framework, FluentResults
DB: SQL Server
Auth: ASP.NET Identity + JWT
Payments: Stripe
Email: AWS SES
Notifications: Telegram Bot
Background Jobs: .NET Hosted Services
